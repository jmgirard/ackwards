# M65: Mechanize the precomputed-vignette staleness guard

- **Status:** in-progress
- **Priority:** normal
- **Depends on:** —
- **Principles touched:** —
- **Branch/PR:** m65-vignette-staleness-guard

## Goal

Make an edited `vignettes/*.Rmd.orig` shipped without a `precompute.R` re-run a mechanical
failure (hash-stamp check) instead of a CLAUDE.md prose warning.

## Scope

**In:** `precompute.R` stamps each generated `.Rmd` with the md5 of its `.Rmd.orig`; a
base-R checker script in `tools/`; a skip-aware testthat wrapper; a source-checkout CI step
in `R-CMD-check.yaml`; one full regeneration to land the stamps; CLAUDE.md prose updated to
point at the guard.

**Out:** detecting hand-edits of the *generated* `.Rmd` (output hashes churn with run noise —
M61 lesson; the stamp's "do not edit by hand" comment + the review consistency-gate cover it);
selective per-vignette regeneration in `precompute.R` (unwanted for now — full regen stays the
one path); the upstream re-oracle watch → M66.

## Acceptance criteria

- [ ] `Rscript tools/check-vignette-freshness.R` exits 0 on a fresh tree; after a scratch edit
      to any `.Rmd.orig` (reverted afterward) it exits non-zero naming the stale vignette; it
      also fails on an orphan (a stamped `.Rmd` whose `.Rmd.orig` is missing, or vice versa).
- [ ] All seven precomputed vignettes carry a stamp comment naming their `.Rmd.orig` and its
      md5; the live `ackwards-interpret.Rmd` carries no stamp and is never flagged.
- [ ] The testthat wrapper passes in `devtools::test()` and skips when no `.Rmd.orig` is
      present (evidence: `R CMD check` green — the tarball excludes `.orig`, so the test must
      skip there).
- [ ] `R-CMD-check.yaml` runs the checker from the source checkout before the tarball check;
      the milestone PR's CI run shows the step green.
- [ ] CLAUDE.md's precompute warning names the mechanical guard instead of prose-only.
- [ ] Profile verify clean: `devtools::test()` clean per task; `devtools::check()` clean at
      review (0 err/0 warn; justify NOTEs).

## Tasks

- [ ] T1: `vignettes/precompute.R` — after each knit, insert an HTML comment below the YAML
      header of the generated `.Rmd`: source filename + md5 of the `.Rmd.orig` + "generated by
      vignettes/precompute.R; do not edit by hand".
- [ ] T2: `tools/check-vignette-freshness.R` — base-R only (`tools::md5sum`), pairs every
      `.Rmd.orig` with its `.Rmd`, verifies stamp presence + hash match both directions,
      prints offenders, non-zero exit on any failure.
- [ ] T3: run `Rscript vignettes/precompute.R` once to land the stamps; commit regenerated
      `*.Rmd` + `vignettes/assets/` (full-regen churn is legitimate here — M61 lesson noted).
- [ ] T4: `tests/testthat/test-vignette-freshness.R` — locates the source tree, skips when no
      `.Rmd.orig` files are found, otherwise runs the T2 checker and asserts success.
- [ ] T5: `R-CMD-check.yaml` — add a `Rscript tools/check-vignette-freshness.R` step after
      `setup-r`, before dependency install (base-R only, fails fast).
- [ ] T6: CLAUDE.md — repoint the "un-regenerated edit ships stale output" prose at the guard.

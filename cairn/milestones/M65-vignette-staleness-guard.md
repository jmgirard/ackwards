## Goal

Make an edited `vignettes/*.Rmd.orig` shipped without a `precompute.R` re-run a mechanical
failure (hash-stamp check) instead of a CLAUDE.md prose warning.

## Scope

**In:** `precompute.R` stamps each generated `.Rmd` with the md5 of its `.Rmd.orig`; a
base-R checker script in `tools/`; a skip-aware testthat wrapper; a source-checkout CI step
in `R-CMD-check.yaml`; one full regeneration to land the stamps; CLAUDE.md prose updated to
point at the guard.

**Out:** detecting hand-edits of the *generated* `.Rmd` (output hashes churn with run noise ‚Äî
M61 lesson; the stamp's "do not edit by hand" comment + the review consistency-gate cover it);
selective per-vignette regeneration in `precompute.R` (unwanted for now ‚Äî full regen stays the
one path); the upstream re-oracle watch ‚Üí M66.

## Acceptance criteria

- [x] `Rscript tools/check-vignette-freshness.R` exits 0 on a fresh tree; after a scratch edit
      to any `.Rmd.orig` (reverted afterward) it exits non-zero naming the stale vignette; it
      also fails on an orphan (a stamped `.Rmd` whose `.Rmd.orig` is missing, or vice versa).
- [x] Every precomputed vignette (each `*.Rmd.orig` ‚Äî currently eight) carries a stamp comment
      naming its `.Rmd.orig` and its md5; the live `ackwards-interpret.Rmd` carries no stamp and
      is never flagged.
- [x] The testthat wrapper passes in `devtools::test()` and skips when no `.Rmd.orig` is
      present (evidence: `R CMD check` green ‚Äî the tarball excludes `.orig`, so the test must
      skip there).
- [x] `R-CMD-check.yaml` runs the checker from the source checkout before the tarball check;
      the milestone PR's CI run shows the step green.
- [x] CLAUDE.md's precompute warning names the mechanical guard instead of prose-only.
- [x] Profile verify clean: `devtools::test()` clean per task; `devtools::check()` clean at
      review (0 err/0 warn; justify NOTEs).

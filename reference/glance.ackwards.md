# Glance at an ackwards object

Returns a one-row data frame of top-level model metadata.

## Usage

``` r
# S3 method for class 'ackwards'
glance(x, ...)
```

## Arguments

- x:

  An `ackwards` object.

- ...:

  Ignored.

## Value

A one-row `data.frame`.
